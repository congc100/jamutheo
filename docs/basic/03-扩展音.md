# 03-扩展音

通过在七和弦上三度堆叠，可以产生 9 音、11 音、13 音等扩展音。

某一扩展音是否可用，主要看它是否在对应和弦音的上方大九度：9 音对应根音，11 音对应三音，13 音对应五音。

| 常用七和弦   | 可用扩展音              |
| :----------- | :---------------------- |
| 大七和弦     | 9，#11，13              |
| 小七和弦     | 9，11, 13               |
| 属七和弦     | 9，b9，#9，#11，13，b13 |
| 小大七和弦   | 9，11，13               |
| 半减七和弦   | 9，11，b13              |
| 减七和弦     | 9，11，b13              |
| 增大七和弦   | 9，#11                  |
| 属七挂四和弦 | 9，b9，#9，11，13，b13  |
