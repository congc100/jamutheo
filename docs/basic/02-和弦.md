# 02-和弦

## 三和弦
- 大三和弦：根音 + 大三度 + 纯五度&emsp;`C`
- 小三和弦：根音 + 小三度 + 纯五度&emsp;`Cm / C-`
- 增三和弦：根音 + 大三度 + 增五度&emsp;`Caug / C+`
- 减三和弦：根音 + 小三度 + 减五度&emsp;`Cdim / C°`
- 挂二和弦：根音 + 大二度 + 纯五度&emsp;`Csus2`
- 挂四和弦：根音 + 纯四度 + 纯五度&emsp;`Csus4`

## 七和弦
- 大七和弦：&emsp;&emsp;大三和弦 + 大七度&emsp;`Cmaj7 / CM7 / CΔ7`
- 小七和弦：&emsp;&emsp;小三和弦 + 小七度&emsp;`Cm7 / C-7`
- 属七和弦：&emsp;&emsp;大三和弦 + 小七度&emsp;`C7`
- 小大七和弦：&emsp;小三和弦 + 大七度&emsp;`CmM7 / C-(maj7)`
- 半减七和弦：&emsp;减三和弦 + 小七度&emsp;`Cø7 / C-7(b5)`
- 减七和弦：&emsp;&emsp;减三和弦 + 减七度&emsp;`Cdim7 / C°7`
- 增属七和弦：&emsp;增三和弦 + 小七度&emsp;`C+7 / Caug7`
- 增大七和弦：&emsp;增三和弦 + 大七度&emsp;`C+(maj7)`
- 属七挂四和弦：挂四和弦 + 小七度&emsp;`C7sus4`

## 加音和弦
- 大六和弦：大三和弦 + 大六度&emsp;&emsp;&emsp;&emsp;&emsp;`C6`
- 小六和弦：小三和弦 + 大六度&emsp;&emsp;&emsp;&emsp;&emsp;`Cm6 / C-6`
- 加九和弦：大三和弦 + 大九度&emsp;&emsp;&emsp;&emsp;&emsp;`Cadd9`
- 六九和弦：大三和弦 + 大六度 + 大九度&emsp;`C6/9`

## 延伸和弦
- 属九和弦：&emsp;属七和弦 + 大九度&emsp;&emsp;&emsp;`C9`
- 大九和弦：&emsp;大七和弦 + 大九度&emsp;&emsp;&emsp;`Cmaj9`
- 小九和弦：&emsp;小七和弦 + 大九度&emsp;&emsp;&emsp;`Cm9`
- 属十一和弦：属九和弦 + 纯十一度&emsp;&emsp;`C11`
- 属十三和弦：属十一和弦 + 大十三度&emsp;`C13`

## 变化和弦
- 属七降九和弦：&emsp;属七和弦 + 降九度&emsp;&emsp;&emsp;&emsp;&emsp;`C7(b9)`
- 属七升九和弦：&emsp;属七和弦 + 升九度&emsp;&emsp;&emsp;&emsp;&emsp;`C7(#9)`
- 属七升十一和弦：属七和弦 + 升十一度&emsp;&emsp;&emsp;&emsp;`C7(#11)`
- 属七降十三和弦：属七和弦 + 降十三度&emsp;&emsp;&emsp;&emsp;`C7(b13)`
- 全变化属七和弦：属七和弦 + 多个变化延伸音&emsp;`C7alt`
