# 03-调式转换

## 同主音调式比较
自然大调音阶从主音出发，各级与主音之间的音程依次为：

::: info 自然大调
纯一度、大二度、大三度、纯四度、纯五度、大六度、大七度、纯八度
:::

对于其他调式，可以将大调音阶的音程关系作为基准，把主音固定在同一个音上进行比较。

通过同主音调式比较，可以突出各个调式的**特色音级**，以便于在不同调式之间进行转换或借用和弦，从而丰富和声色彩。

后面对于每种调式所标记的“升”和“降”，都指的是在音程意义上，当前音级相比自然大调的对应音级升高或降低了半音。

## 四种常见调式对比
- 自然大调：1，2，3，4，5，6，7，1
- 自然小调：1，2，**b3**，4，5，**b6**，**b7**，1
- 和声小调：1，2，**b3**，4，5，**b6**，7，1
- 旋律小调：1，2，**b3**，4，5，6，7，1
