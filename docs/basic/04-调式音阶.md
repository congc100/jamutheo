# 04-调式音阶

自然大调音阶从主音出发，各级与主音之间的音程依次为：

::: info 自然大调
纯一度、大二度、大三度、纯四度、纯五度、大六度、大七度、纯八度
:::

对于其他调式，可以将大调音阶的音程关系作为基准，把主音固定在同一个音上进行比较。

通过同主音调式比较，可以突出各个调式的**特色音级**，以便于在不同调式之间进行转换或借用和弦，从而丰富和声色彩。

后面对于每种调式所标记的“升”和“降”，都指的是在音程意义上，当前音级相比自然大调的对应音级升高或降低了半音。

## 核心大小调对比

- 自然大调：1，2，3，4，5，6，7，1
- 自然小调：1，2，**b3**，4，5，**b6**，**b7**，1
- 和声小调：1，2，**b3**，4，5，**b6**，7，1
- 旋律小调：1，2，**b3**，4，5，6，7，1

## 关系调视角下的调式音阶

按照 C 大调中七个音的顺序，分别以每个音作为首音构建音阶，即可得到七种调式。

- 伊奥尼亚（Ionian）：以 do 为首音，即自然大调本身
- 多利亚（Dorian）：以 re 为首音
- 弗里吉亚（Phrygian）：以 mi 为首音
- 利底亚（Lydian）：以 fa 为首音
- 混合利底亚（Mixolydian）：以 sol 为首音
- 爱奥利亚（Aeolian）：以 la 为首音，即自然小调本身
- 洛克利亚（Locrian）：以 si 为首音

## 同主音视角下的调式音阶

将七种调式都建立在同一个主音上，可以根据根音与三音构成大三度还是小三度，将它们分为大调类和小调类。

大调类

- 伊奥尼亚：&emsp;1，2，3，4，5，6，7
- 利底亚：&emsp;&emsp;1，2，3，**#4**，5，6，7
- 混合利底亚：1，2，3，4，5，6，**b7**

小调类

- 爱奥利亚：&emsp;1，2，b3，4，5，b6，b7
- 多利亚：&emsp;&emsp;1，2，b3，4，5，**6**，b7
- 弗里吉亚：&emsp;1，**b2**，b3，4，5，b6，b7
- 洛克利亚：&emsp;1，b2，b3，4，**b5**，b6，b7
