# 11-惯用线

惯用线是一种制造和声运动错觉的技巧。其原理是：在和弦根音与三音保持不变的前提下，某一声部做半音级进。这会使和弦名称和色彩发生改变，但和声的实际功能并未转换。

根据运动声部所处的位置，会产生不同的听觉效果：

- 高音部：常用于编曲中的主旋律伴奏。
- 内声部：常用于键盘或吉他编曲中的和声填充。
- 低音部：常用于编写贝斯线，产生斜向运动。

运动声部通常在和弦的六度与七度音之间作半音移动，实际应用中非常灵活，既可走完全程，也可只走两三步。

- 下行惯用线：大七度 → 小七度 → 大六度 → 小六度 (B → Bb → A → Ab)。
  - 小调示例：Cm(maj7) → Cm7 → Cm6 → Cm(b6)
  - 大调示例：Cmaj7 → C7 → C6 → C(b6)
- 上行惯用线：下行轨迹的逆向运用，从小六度半音级进爬升至大七度 (Ab → A → Bb → B)。
