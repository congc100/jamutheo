# 11-小调和声

小调中的和声功能与大调相似，同样存在自然功能和弦、副属和弦、相关2级和弦、调式转换和弦等。

## 小调自然和弦

在现代音乐中，一般将自然小调、和声小调、旋律小调、多利亚调式综合在一起，作为“小调”自由使用，这四个调式中的音都被视为自然音。

小调在各个级数可用的调内自然和弦总结如下：

| 级数 |     C 自然小调      |       C 和声小调       |       C 旋律小调       |    C 多利亚调式     |
| :--: | :-----------------: | :--------------------: | :--------------------: | :-----------------: |
|  1   |   Cm7<br>小七和弦   | Cm(maj7)<br>小大七和弦 | Cm(maj7)<br>小大七和弦 |   Cm7<br>小七和弦   |
|  2   | Dm7b5<br>半减七和弦 |  Dm7b5<br>半减七和弦   |    Dm7<br>小七和弦     |   Dm7<br>小七和弦   |
| 3/b3 | Ebmaj7<br>大七和弦  | Ebmaj7#5<br>增大七和弦 | Ebmaj7#5<br>增大七和弦 | Ebmaj7<br>大七和弦  |
|  4   |   Fm7<br>小七和弦   |    Fm7<br>小七和弦     |     F7<br>属七和弦     |   F7<br>属七和弦    |
|  5   |   Gm7<br>小七和弦   |     G7<br>属七和弦     |     G7<br>属七和弦     |   Gm7<br>小七和弦   |
| 6/b6 | Abmaj7<br>大七和弦  |   Abmaj7<br>大七和弦   |  Am7b5<br>半减七和弦   | Am7b5<br>半减七和弦 |
| 7/b7 |   Bb7<br>属七和弦   |   Bdim7<br>减七和弦    |  Bm7b5<br>半减七和弦   | Bbmaj7<br>大七和弦  |

## 小调 II → V → I 进行

参考小调调内自然和弦，可以基于和声小调构建最基本的 II → V → I 进行：

- 大调：II-7 → V7 → Imaj7
- 小调：II-7(b5) → V7(b9) → I-(maj7)

在此基础上，进行和大调类似的同功能和弦替代等操作，同样可以衍生出多种变体。
