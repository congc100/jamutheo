# 10-减七和弦

减七和弦包含两个三全音，极不稳定，强烈倾向于进行到一个稳定的和弦，主要用于过渡连接。
- 常见的解决方式是根音小二度上行或下行，可作为经过和弦使用，添加到相邻级数的进行中间。
- 也可在保持低音不变的前提下，作为辅助和弦使用，提供调外听感。

减七和弦可以用替代属七降九和弦，因为减七和弦的组成音与无根音的属七降九和弦相同。

减七和弦之间可以相互替代，根据实际组成音划分，仅有三种减七和弦：
- 第一组：C°7，bE°7，bG°7，A°7
- 第二组：bD°7，E°7，G°7，bB°7
- 第三组：D°7，F°7，bA°7，bC°7

根音到三音、三音到五音、五音到七音、七音到高八度根音，均相差三个半音，这种对称性使得和弦中的任何一个音都可被视为根音。
