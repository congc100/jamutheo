# 08-调式转换和弦功能

最常见的调式转换，是在大调中借用小调和弦。下表以 C 自然大调与 C 自然小调为例，对比同主音大小调自然七和弦的性质：

| 级数 | C 自然大调          | C 自然小调          |
| :--: | :-----------------: | :-----------------: |
| 1    | Cmaj7<br>大七和弦   | Cm7<br>小七和弦     |
| 2    | Dm7<br>小七和弦     | Dm7b5<br>半减七和弦 |
| 3/b3 | Em7<br>小七和弦     | Ebmaj7<br>大七和弦  |
| 4    | Fmaj7<br>大七和弦   | Fm7<br>小七和弦     |
| 5    | G7<br>属七和弦      | Gm7<br>小七和弦     |
| 6/b6 | Am7<br>小七和弦     | Abmaj7<br>大七和弦  |
| 7/b7 | Bm7b5<br>半减七和弦 | Bb7<br>属七和弦     |

## 小调借用和弦的功能划分

同主音小调中的七个级数按功能可分为以下两类：
- 主功能和弦：1 级，b3 级，5 级
- 非主功能和弦：2 级，4 级，b6 级，b7 级

区分这两类功能的关键在于特色音（b6 音）的有无。
- 主功能和弦均不含小调特色音 b6；
- 非主功能和弦均含 b6，该音在大调中相当于4级和弦的降三音，因此这些和弦都被视为具有**下属功能**。

值得注意的是，b7 级属七和弦虽然结构上为大小七和弦的结构，但在功能上并不具有属功能。一种常见情况是，b7 级属七和弦进行到 b3 级大七和弦，由于 b3 级大七并非对应大调中的自然和弦，因此这一进行应被理解为假解决。

## 常见的借用和弦

### 主功能小调和弦

1 级小七和弦
- 添加扩展音 9，11，13，可形成多利亚音阶

b3 级大七和弦
- 添加扩展音 9，#11，13，可形成利底亚音阶

5 级小七和弦
- 添加扩展音 9，11，13，可形成多利亚音阶

### 下属小调和弦

2 级半减七和弦
- 添加扩展音 9，11，b13，可形成洛克利亚还原九

4 级小七和弦
- 添加扩展音 9，11, 13，可形成多利亚音阶

b6 级大七和弦
- 添加扩展音 9，#11，13，可形成利底亚音阶

b7 级属七和弦
- 添加扩展音 9，#11，13，可形成混合利底亚升十一

### 变化下属小调和弦

b2 级大七和弦
- 可视为 2 级半减七和弦把根音降半音，也可视为弗里吉亚调式的 b2 级大七和弦
- 以 b2 为新主音，添加扩展音 9，#11，13，可形成利底亚音阶

4 级小大七和弦
- 可视为 4 级小七和弦把七音升半音
- 添加扩展音 9，11，13，可形成旋律小调音阶

b6 级属七和弦
- 可视为 b6 级大七和弦把七音降半音
- 添加扩展音 9，#11，13，可形成混合利底亚升十一

b7 级大七和弦
- 可视为 b7 级属七和弦把七音升半音，也可视为混合利底亚调式的 b7 级大七和弦
- 以 b7 为新主音，添加扩展音 9，#11，13，可形成利底亚音阶

### 来自其他调式的借用和弦

5 级属七降九和弦
- 可视为借用自和声小调的 5 级九和弦

6 级半减七和弦
- 可视为借用自旋律小调的 6 级和弦
- 添加扩展音 9，11，b13，可形成洛克利亚还原九
