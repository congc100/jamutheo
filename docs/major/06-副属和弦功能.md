# 06-副属和弦功能

## 属和弦

属功能和弦可以向下五度解决到任意性质的调内自然和弦（减三和弦除外），其中：

- 解决到 1 级的 5 级和弦称为主属和弦；
- 解决到 2~6 级的“相对 5 级”和弦则称为副属和弦，记作“V/x”，x 为目标和弦的级数。

由于副属和弦含有调外的变化音，使用时须满足两个条件：

- 根音必须是调内自然音；
- 解决的目标也必须是调内自然和弦。

若属和弦没有按照向下纯五度的关系解决到预期的目标和弦，则称为假解决。

| 副属和弦 |               音阶构成                |       和弦音阶        |
| :------: | :-----------------------------------: | :-------------------: |

|  V7/IV   |      1，2，3，4，5，6，**b7**，1      |      混合利底亚       |
|   V7/V   |      1，2，3，4，5，6，**b7**，1      |      混合利底亚       |
|  V7/VI   | 1，**b2**，3，4，5，**b6**，**b7**，1 | 混合利底亚降九/降十三 |

## 副属和弦扩展音

副属和弦中扩展音的使用原则与属七和弦基本一致，但更偏向使用自然音。

- 9，b9，#9，13，b13 均可使用；
- 11 不可用，#11 一般不用；
- 若 #9 是自然音，则无论 b9 是否为自然音，都可以同时使用。

| 副属和弦 | 自然扩展音 | #9 是自然音时的扩展音 |
| :------: | :--------: | :-------------------: |
|  V7/II   |   9，b13   |      b9，#9，b13      |
|  V7/III  |  b9，b13   |      b9，#9，b13      |
|  V7/IV   |   9，13    |        不可能         |
|   V7/V   |   9，13    |      b9，#9，13       |
|  V7/VI   |  b9，b13   |      b9，#9，b13      |
