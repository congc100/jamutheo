# 06-属和弦功能

## 属和弦

属功能和弦可以向下五度解决到任意性质的调内自然和弦（减三和弦除外），其中：

- 解决到 1 级的 5 级和弦称为主属和弦；
- 解决到 2~6 级的“相对 5 级”和弦则称为副属和弦，记作“V/X”，X 为目标和弦的级数。

由于副属和弦含有调外的变化音，使用时须满足两个条件：

- 根音必须是调内自然音；
- 解决的目标也必须是调内自然和弦。

若属和弦没有按照向下纯五度的关系解决到预期的目标和弦，则称为假解决。

## 替代属和弦

属七和弦的解决实际上主要是三全音的解决。将原本的三音和七音进行转位，可以得到等音程的新的三音和七音，形成新的属七和弦。

这种三全音相同的属七和弦之间的替代称为三全音替代，转位得到的新属七和弦称为替代属和弦。

## 属和弦扩展音

属和弦的音阶基础是混合利底亚，避免使用 11 音，因为会和 3 音形成小二度。

主属和弦的扩展音可以有多种选择，从单个变化音到 Alt 属和弦均可。

副属和弦中扩展音的使用原则与属七和弦基本一致，但最好多用原调的自然音。

| 副属和弦 |          可选适配音阶          |          音阶构成           |
| :------: | :----------------------------: | :-------------------------: |
|  V7/IV   |           混合利底亚           |   1，2，3，4，5，6，b7，1   |
|   V7/V   |           混合利底亚           |   1，2，3，4，5，6，b7，1   |
|  V7/II   |        混合利底亚降十三        |  1，2，3，4，5，b6，b7，1   |
|  V7/III  | 混合利底亚<br>降九/升九/降十三 | 1，b2，#2，3，b5，b6，b7，1 |
|  V7/VI   | 混合利底亚<br>降九/升九/降十三 | 1，b2，#2，3，b5，b6，b7，1 |

备注：

- 对于 V7/II，b13 音会与 5 音形成小二度，需要避免二者同时出现
