# 10-减七和弦

减七和弦包含两个三全音，极不稳定，强烈倾向于进行到一个稳定的和弦，主要用于过渡连接。

- 常见的解决方式是根音小二度上行或下行，可作为经过和弦使用，添加到相邻级数的进行中间。
- 也可在保持低音不变的前提下，作为辅助和弦使用，提供调外听感。

## 减和弦的转化

减七和弦可以用于替代属七降九和弦，因为减七和弦的组成音与无根音的属七降九和弦相同。

减七和弦之间可以相互替代，根据实际组成音划分，仅有三种减七和弦：

- 第一组：C°7，bE°7，bG°7，A°7
- 第二组：bD°7，E°7，G°7，bB°7
- 第三组：D°7，F°7，bA°7，bC°7

根音到三音、三音到五音、五音到七音、七音到高八度根音，均相差三个半音，这种对称性使得和弦中的任何一个音都可被视为根音。

## 减和弦的走向

减七和弦一般都要明确地走到自己的目标和弦上，即使需要变化解决目标，低音仍然要保持半音移动。

| 减七和弦 | 期望解决目标 |   变化解决目标   |
| :------: | :----------: | :--------------: |
| #1 和 b3 |   2 级小七   |  5 级属七 on 2   |
|    #2    |   3 级小七   |  1 级大七 on 3   |
| #4 和 b6 |   5 级属七   |  1 级大七 on 5   |
|    #5    |   6 级小七   | 6 级属七 (V7/II) |

## 减和弦扩展音

按照对应和弦音上方大二度的规则，减七和弦的扩展音共有四个：9 音，11 音，b13 音，maj7 音。

将减七和弦的和弦音和扩展音放在一起，形成的音阶是由全音和半音交替构成的，将其称作全半减音阶。

如果是以功能和声的方式使用减七和弦，最好只使用贴合解决目标所在调式的扩展音。

以 #1 级减七到 2 级小七为例：

- b9 音是自然音（2 级），但与根音（#1 级）是小二度关系，不可用；
- b11 音是自然音（4 级），但与三音（3 级）是小二度关系，不可用；
- b13 音是自然音（6 级），与五音（5 级）是大二度关系，可用；
- maj7 音是自然音（1 级），与七音（b7 级）是大二度关系，可用。

## 减音阶

全半减音阶由全音和半音交替构成，具备对称性。将其左右反转，形成的音阶称作半全减音阶。

相比全半减音阶，半全减音阶没有可用扩展音，因为和弦音上方二度均为小二度。

半全减音阶更多的是用在属和弦上，称为合成属音阶，前半段可以视为变化属音阶，后半段可以视为利底亚降七。

| 音阶       | 构成                               |
| ---------- | ---------------------------------- |
| 变化属音阶 | <u>1，b2，#2，3</u>，b5，b6，b7，1 |
| 半全减音阶 | 1，b2，#2，3，#4，5，6，b7，1      |
| 利底亚降七 | 1，2，3，<u>#4，5，6，b7，1</u>    |
